# Игра "Падающие мячики"
Было создана мною, Zerok0, во время прохождения курса разработки на Python, как показатель моего знания языка.
В нем используется связка pygame и pymunk для создания физики падающих мячиков и их картинок.

## Инструкция по установке
1. Убедитесь, что у вас настроено виртуальное пространство
2. Пропишите `pip install requirements.txt`
3. Убедитесь, что у вас установлен Python
4. Запустите main.py
5. Готово
